# GLSL Fractal Visualizations

Interactive WebGL fractal visualizations with fluid flow effects.

## Features

- **3D Sphere Display**: Hyperbolic, Parabolic, and Elliptical geometry modes
- **Fluid Flow Effects**: Organic, flowing distortions with adjustable intensity, speed, and viscosity
- **Interactive Controls**: 
  - Time speed, zoom, curvature, iterations
  - Brightness (internal lighting)
  - Color schemes (monochromatic, analogous, complementary, triadic, etc.)
  - Camera rotation (Shift + drag to rotate view)
  - Grid rotation (drag to rotate pattern)

## Files

- `glsl-fluid-fractals.html` - Main interactive fractal visualization

## Usage

Open `glsl-fluid-fractals.html` in a modern web browser with WebGL support.

## Controls

- **Mouse drag**: Rotate the fractal pattern
- **Shift + Mouse drag**: Rotate camera view
- **Control panel**: Adjust all parameters in real-time
